# How to Install WordPress

Installing WordPress on Znode is very easy using Softaculous, our automatic installer.

---

## Installation Steps

1. Log in to your control panel (via client area)
2. Go to **Softaculous Apps Installer**
3. Click **WordPress**
4. Click **Install Now**
5. Fill in the settings:
   - Choose your domain
   - Set admin username and password
   - Enter site name and description
6. Click **Install**

---

## That's It!

WordPress is now installed. You can access:

| Destination | URL |
|-------------|-----|
| Your website | `http://yourdomain.com` |
| Admin area | `http://yourdomain.com/wp-admin` |

Links are also shown on the installation success page.

---

## Recommended Next Steps

After installation, consider:

1. **Change permalink structure:** Settings → Permalinks → choose "Post name"
2. **Install essential plugins:** Security, caching, SEO
3. **Choose a theme:** Appearance → Themes
4. **Set up HTTPS:** Get a free SSL certificate
5. **Create your first content:** Posts or Pages

---

## Installation Options

During installation, you can customize:

| Setting | Description |
|---------|-------------|
| Installation URL | Your domain and optional subfolder |
| Site name | Your website's title |
| Site description | Tagline shown in some themes |
| Admin username | Your login name (avoid "admin") |
| Admin password | Strong password for security |
| Admin email | For notifications and password recovery |

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Installation fails | Check disk space and inode limits |
| Can't access admin | Clear browser cache, try incognito mode |
| Database error | Verify MySQL credentials in wp-config.php |
